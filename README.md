# Docker

Структура проекта:
my_app/
├── app/
│   ├── main.py
│   ├── requirements.txt
│   └── Dockerfile
├── docker-compose.yml
└── .env

main.py: приложение.
requirements.txt: Список зависимостей Python.
Dockerfile: Для создания Docker-образа.
docker-compose.yml: Конфигурация для запуска контейнеров.
.env: Файл для переменных окружения.
